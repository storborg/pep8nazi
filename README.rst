==================================================================================
pep8nazi - Because it's 2011 people.
==================================================================================

:Authors:
    Scott Torborg (storborg)
:Version: 0.1

This library prevents non-PEP8 code from being imported into the interpreter.

Installation
============

Simple as::

    $ easy_install pep8nazi 

Or if you prefer:

    $ pip install pep8nazi
